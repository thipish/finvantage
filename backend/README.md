# FinVantage ML Backend

## Quick Start

### 1. Install dependencies
```bash
cd backend
pip install -r requirements.txt
```

### 2. Set up MySQL
```bash
mysql -u root -p < sql/schema.sql
```

### 3. Train the model
```bash
python train_model.py
```

### 4. Configure environment variables
```bash
export DATABASE_URL="mysql+pymysql://your_user:YOUR_STRONG_PASSWORD@localhost:3306/finvantage"
export FINVANTAGE_API_KEY="$(openssl rand -hex 32)"
export JWT_SECRET="$(openssl rand -hex 32)"
export ALLOWED_ORIGINS="http://localhost:5173,https://finvantage.lovable.app"
```

### 5. Start the API server
```bash
uvicorn app:app --reload --port 8000
```

### 6. Test endpoints

**Register:** `POST /api/auth/register` (fullName, email, password)
**Login:** `POST /api/auth/login` (email, password) → returns JWT
**Predict:** `POST /api/predict` (requires Bearer token + X-API-Key)
**History:** `GET /api/history` (requires Bearer token + X-API-Key)

## Architecture
```
React → /api/auth/* → FastAPI → bcrypt + JWT → MySQL
React → /api/predict → FastAPI → Random Forest .pkl → MySQL → Response
React → /api/history → FastAPI → MySQL → Assessment History
```

## Files
- `sql/schema.sql` — MySQL tables (auth_users, users, loan_applications, risk_predictions, assessment_history)
- `data/sample_historical_loan_data.csv` — Training dataset
- `train_model.py` — Model training script (Random Forest)
- `app.py` — FastAPI server with auth, prediction, and history endpoints
- `models/` — Exported .pkl artefacts (generated by train_model.py)
